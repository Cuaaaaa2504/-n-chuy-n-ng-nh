# P45 – Danh sách phim + tìm kiếm/lọc

Đã bổ sung:
- `vite.config.js`.
- Cài và dùng `react-router-dom`; nút **Xem chi tiết** dùng `Link`.
- Dùng `axiosClient` thống nhất kiểu P42 thay vì apiClient rời rạc.
- Thêm phân trang client/API qua `page` và `limit`.

Chạy:
```bash
npm install
copy .env.example .env
npm run dev
```
