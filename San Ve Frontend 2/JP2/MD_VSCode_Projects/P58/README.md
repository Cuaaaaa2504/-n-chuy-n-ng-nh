# P58 – Vé của tôi + modal QR

Đã bổ sung:
- `vite.config.js`.
- Dùng `react-router-dom` thật trong `App.tsx`.
- Nút **Hủy đơn** và link **Quay lại thanh toán** cho `PENDING_PAYMENT`.
- Modal đóng bằng click ra ngoài hoặc phím `Esc`.
- Phân trang danh sách booking.

Chạy:
```bash
npm install
copy .env.example .env
npm run dev
```
